# Test

## 관련 모듈

### Jest

- <https://jestjs.io/>
- 테스트 프레임워크

### SuperTest

- <https://www.npmjs.com/package/supertest>
- HTTP 메소드(GET/POST/PUT... 등) 요청 관련 테스트

## 설치

``` bash
npm install --save-dev jest supertest
```
